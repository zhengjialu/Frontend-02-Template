> 找出 JavaScript 标准里面所有具有特殊行为的对象

```javascript
// 这块知识比较薄弱，我理解的特殊行为是改变属性的内部方法。

Object: 
	[[prototype]]：__proto__ 原型链
	[[SetPrototypeOf]]：给对象设置一个指定的新原型
  [[GetPrototypeOf]]：获取指定对象的原型
  [[Get]]：属性调用时执行特定方法
  [[Set]]：属性设置时执行特定方法
	[[Delete]]：删除指定属性
	[[DefineOwnProperty]]：定义属性内部的状态
  [[Enumerate]]：属性是否可枚举
  [[Writable]]：属性是否可写
  [[Configurable]]：属性是否可更改
	[[GetOwnProperty]]
	[[GetProperty]]
	[[CanPut]]
	[[Put]]
	[[HasProperty]]
	[[DefaultValue]]
	[[Value]]
Array: 
	[[length]]：数组可变长度
  [[DefineOwnProperty]]
Function: 
	[[call]]：调用其他函数的 thisObj
  [[Construct]]
	[[ThrowTypeError]]
	[[HasInstance]]
	[[Get]]
String:
	[[GetOwnProperty]]
```
